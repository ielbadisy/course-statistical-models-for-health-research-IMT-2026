
# Statistical Models for Health Research (IMT Brest, 2026)

Introductory course on regression modeling for clinical and epidemiological research using R.

Focus:

* Generalized Linear Models

  * Linear regression

  * Logistic regression

* Survival analysis (Cox proportional hazards model and extension to survdnn)

* Model diagnostics and interpretation in health research


## Structure

* `glms-application-student.qmd`: Linear and Logistic regression lab

* `survival-cox-application-students.qmd`: Cox and beyond lab

* `project-treatment-comparison-melanoma.qmd`: Applied course project

* `datasets/`: Course datasets
